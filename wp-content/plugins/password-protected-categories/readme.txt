Password protect your blog categories and custom taxonomies. Unlock them with a simple password.

== Description ==

The only WordPress plugin to password protect entire categories of blog posts, products, events, portfolios, or other types of website content.

== Installation ==

1. Go to Plugins -> Add New -> Upload and select the plugin ZIP file (see link in Purchase Confirmation Email).
1. Activate the plugin.
1. Enter your license key under Settings -> Protected Categories.
1. Create or edit any category or custom taxonomy, e.g. under the Posts -> Categories menu.
1. Set the 'Visibility' to Password Protected and enter one or more passwords.
1. The category (and all of its posts) will now be password protected.
